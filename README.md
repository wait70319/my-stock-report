# 📊 全球股市匯市自動化日報 (美、日、台)

這是一個自動化系統，每天定時抓取最新的市場數據與財經新聞，並生成視覺化網頁報告。

## 🌐 報告網址
[點此查看最新日報](https://您的帳號.github.io/my-stock-report/)
*(請將上面的網址換成您在 Settings > Pages 看到的網址)*

---

### 🚀 系統功能
* **自動抓取**：每日清晨自動獲取 S&P 500、台股加權、日經 225 及關鍵匯率。
* **AI 投資分析**：整合近期市場趨勢，追蹤未來 5 年具高需求潛力的標的。
* **零成本託管**：使用 GitHub Actions + Pages，完全免費執行。

---

### 📈 近期投資與未來 5 年核心配置
根據 2026 年初的市場邏輯，本報表重點追蹤以下領域：

#### 1. 近期標的 (Short-Term Focus)
* **美股**：AI 應用軟體與大型雲端供應商 (Microsoft, Amazon)。
* **台股**：半導體先進封裝 (CoWoS) 與水冷散熱供應鏈。
* **日股**：掌握全球核心技術的半導體材料商。

#### 2. 未來 5 年高成長領域 (5-Year Growth)
* **能源基礎建設**：AI 資料中心驅動的核能與電力網轉型。
* **生技醫療**：人口老化推升的長壽醫療與 AI 藥物篩選。
* **國防與太空**：低軌衛星通訊與防衛系統自動化。

---

### 🛠️ 技術說明
* **語言**: Python 3.9+
* **數據來源**: Yahoo Finance (yfinance)
* **新聞來源**: Google News RSS
* **自動化**: GitHub Actions (排程設定為每天台灣時間 06:00)
